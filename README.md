# lilOG

Open Graph Preview and Inspector

Scan any live URL and preview its Open Graph cards on Facebook, X, LinkedIn, WhatsApp, and Discord. Tweak the tags, inspect them, and copy a clean head block.

Part of [Hot App Summer](https://hotappsummer.netlify.app) by [lilAgents](https://lilagents.com): one small, useful web tool shipped every day through June.

## Run it locally

This is an [Astro](https://astro.build) site.

```bash
pnpm install
pnpm dev
```

To build for production:

```bash
pnpm build
```
